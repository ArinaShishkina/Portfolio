# Arina Portfolio Website

A personal portfolio website built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Publish on GitHub

1. Push this project to a GitHub repository on the `main` branch.
2. In GitHub, open **Settings -> Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to `main` again (or run the workflow manually from **Actions**).
5. Your site will be deployed automatically.

## Update your content

- Main page structure: `src/App.jsx`
- Global styles: `src/index.css`
- Replace placeholder videos by editing `PASTE_YOUR_VIDEO_URL_HERE` in `src/App.jsx`
