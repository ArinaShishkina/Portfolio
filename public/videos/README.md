# Local videos

Put your video files in this folder.

Then set each `src` in `src/App.jsx` like:

- `/videos/emerald-brand-project.mp4`
- `/videos/ucd-smurfit-interview-reel.mp4`
- `/videos/your-new-video.mp4`

Tips:

- Use `.mp4` (H.264) for best browser support.
- Keep filenames lowercase and use `-` instead of spaces.
